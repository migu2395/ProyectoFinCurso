## 📌 Descripción

**OPNsense** es un sistema operativo basado en **FreeBSD**, diseñado para funcionar como **router y firewall**. Surge como una bifurcación de **pfSense** y utiliza el firewall **Packet Filter (PF)** de FreeBSD.

Está orientado tanto a **entornos domésticos** como **empresariales**, ofreciendo una solución robusta, flexible y segura.

> Se utilizará la última versión estable: **26.1**

---

## ⚙️ Características principales

OPNsense proporciona:

- 🔥 Firewall avanzado  
- 🌐 Routing: NAT, VLAN y multi-WAN  
- 🧭 DNS mediante **UnboundDNS**  
- 🛡️ IDS con **Suricata**  
- 🔐 Acceso remoto con **Tailscale**  
- 📊 Monitorización en tiempo real con **NetFlow**  
- 🧩 Amplio sistema de plugins  

---

## ⚖️ Comparativa con alternativas

Frente a otras soluciones como **pfSense** y **OpenWRT**, OPNsense destaca por:

- 🔄 Actualizaciones más frecuentes  
- 🖥️ Interfaz más accesible  
- 🧩 Sistema de plugins más flexible  
- 🔐 Seguridad más estricta  
- 🔗 Mejor integración con otras tecnologías  
- 📊 Monitorización más completa  
- 🆓 Totalmente open source
